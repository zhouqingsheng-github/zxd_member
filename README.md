# 酒店会员小程序

这是一个酒店 C 端小程序，提供酒店预订、会员服务等功能。

## 主要功能

### 1. 首页
- 酒店搜索
- 热门酒店推荐
- 优惠活动展示
- 会员特权展示

### 2. 订单管理
- 订单列表查看
- 订单详情
- 订单状态跟踪
- 订单评价

### 3. 会员中心
- 会员等级展示
- 会员权益说明
- 积分查询
- 会员卡管理

### 4. 个人中心
- 个人信息管理
- 常用旅客管理
- 发票管理
- 设置

## 技术栈
- 前端：微信小程序
- 后端：Node.js
- 数据库：MySQL

## 项目结构
```
├── pages/              # 页面文件
│   ├── home/          # 首页
│   ├── order/         # 订单页
│   ├── member/        # 会员页
│   └── mine/          # 我的页面
├── components/        # 组件
├── static/            # 静态资源
│   ├── images/        # 图片资源
│   └── icons/         # 图标资源
└── utils/             # 工具函数
```

## 安装和运行
1. 克隆项目
2. 安装依赖
3. 使用微信开发者工具打开项目
4. 编译运行

## 注意事项
- 需要配置小程序的 AppID
- 需要配置后端接口地址
- 需要配置相关 API 密钥
